L'ARENA DEL LIMBO

Nessun labirinto. Nessuna porta da aprire. L'intero gioco ti confina in un'immensa distesa di rovine antiche e spoglie, sospese nel vuoto assoluto del Limbo.

Colonne spezzate e pavimenti di pietra consumati dal tempo sono l'unica copertura contro l'orrore. Non c'è un posto dove scappare, ci sei solo tu e l'onda. Più combatti, più l'ambiente reagisce: i cieli si squarciano, la nebbia muta, e le pareti iniziano a trasudare sangue.

I DIECI CANCELLI

I rimorsi e le colpe che devi respingere per evitare di essere consumato.

Ondate I - III

I Primi Peccati

Nebbia grigia e silenzio spettrale. I nemici primordiali (Bugie e Viltà) strisciano veloci fuori dalle ombre dei pilastri. Ritrovi la tua Pistola e il Fucile tra le macerie.

Ondate IV - VI

Il Peso della Colpa

La nebbia si tinge di un rosso denso. Appaiono i Tank (l'Egoismo), colossi imponenti che frantumano le colonne di pietra mentre caricano. Lo spazio vitale dell'arena si riduce.

Ondate VII - IX

L'Asfissia Spirituale

Il caos strutturale è totale. Le mura piangono sangue. Sblocchi lo Shotgun e l'SMG per sopravvivere a orde iper-aggressive generate da ogni angolo cieco.

Ondata X

Il Banchetto Finale

Silenzio tombale. Il cielo si spezza in frammenti. Malachar discende al centro della mappa: un'entità colossale avvolta in un mantello di anime e protetta da una maschera d'oro sacro.

using System.Collections;
using UnityEngine;

public class WaveManager : MonoBehaviour 
{
    private int currentGate = 1;
    public EnvironmentController envController;

    public void CheckWaveProgress(int enemiesRemaining) 
    {
        if (enemiesRemaining == 0) 
        {
            currentGate++;
            if (currentGate == 4) envController.SetSkyColor(Color.red);
            if (currentGate == 7) envController.TriggerBleedingWalls();
            if (currentGate == 10) SpawnMalachar();
            else StartCoroutine(SpawnNextWave());
        }
    }
}
using UnityEngine;

public class MalacharBoss : EnemyBase, IDamageable
{
    private bool maskBroken = false;
    public GameObject goldenMaskModel;
    public ParticleSystem soulsCloak;

    public override void TakeDamage(float dmg) 
    {
        health -= dmg;
        if (health <= maxHealth / 2 && !maskBroken) 
        {
            ShatterGoldenMask();
        }
        if (health <= 0) TriggerEpilogue();
    }

    private void ShatterGoldenMask() 
    {
        maskBroken = true;
        goldenMaskModel.SetActive(false);
        // Il cielo si apre, l'arena trema
        CameraShake.Instance.Shake(5f);
    }
}
